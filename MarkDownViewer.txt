# MarkdownViewer Project Setup

This document outlines the steps to create a standalone Markdown viewer project.

## 1. Create the Project Directory

Create a new directory for the project:
`mkdir E:\Mema-Lab\MarkdownViewer`

## 2. Create Project Files

Navigate into the new directory:
`cd E:\Mema-Lab\MarkdownViewer`

Create the following files with the specified content:

### 2.1. `README.md`

```markdown
# MarkdownViewer

A simple Python script to convert Markdown files to HTML and open them in a web browser.

## Usage:

```bash
python viewer.py <path_to_markdown_file>
```
```

### 2.2. `requirements.txt`

```
markdown
```

### 2.3. `viewer.py`

```python
import markdown
import webbrowser
import os
import sys

def convert_md_to_html(md_file_path):
    """Converts a Markdown file to HTML."""
    with open(md_file_path, 'r', encoding='utf-8') as f:
        md_content = f.read()
    html_content = markdown.markdown(md_content)
    return html_content

def save_html_and_open(html_content, original_md_path):
    """Saves HTML content to a temporary file and opens it in a browser."""
    # Create a temporary HTML file name based on the Markdown file
    base_name = os.path.basename(original_md_path)
    file_name_without_ext = os.path.splitext(base_name)[0]
    temp_html_file = f"{file_name_without_ext}.html"
    temp_html_path = os.path.join(os.path.dirname(original_md_path), temp_html_file)

    with open(temp_html_path, 'w', encoding='utf-8') as f:
        f.write(html_content)

    print(f"Generated HTML file: {temp_html_path}")
    webbrowser.open(f"file:///{os.path.abspath(temp_html_path)}")

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python viewer.py <path_to_markdown_file>")
        sys.exit(1)

    markdown_file_path = sys.argv[1]

    if not os.path.exists(markdown_file_path):
        print(f"Error: File not found at {markdown_file_path}")
        sys.exit(1)

    try:
        html_output = convert_md_to_html(markdown_file_path)
        save_html_and_open(html_output, markdown_file_path)
    except Exception as e:
        print(f"An error occurred: {e}")
        sys.exit(1)
```

## 3. Install Dependencies

From within the `E:\Mema-Lab\MarkdownViewer` directory, install the required Python packages:
`pip install -r requirements.txt`

## 4. Usage

To view a Markdown file, run the `viewer.py` script from the `E:\Mema-Lab\MarkdownViewer` directory, providing the path to the Markdown file you wish to view:
`python viewer.py <path_to_your_markdown_file.md>`
